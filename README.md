# AI Commerce OS

AI-powered ecommerce dashboard built using Next.js and React.

## Features

- CEO Dashboard
- AI Business Assistant
- Product Catalog
- Shopping Cart
- Checkout Workflow
- Order Summary
- Order Confirmation
- Business Analytics Simulation

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
